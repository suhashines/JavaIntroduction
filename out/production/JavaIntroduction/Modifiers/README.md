<h1 align="center">Java Modifiers</h1>


✔ **Public**

- Accessible everywhere.

✔ **Private**

- Only inside the same class.

✔ **Default (package-private)**

- Accessible only inside same package.

## After Learning Inheritance

✔ **Protected**

- Accessible in same package

- Accessible in subclasses even if different package

✔ **Final**

- final class → cannot inherit

- final method → cannot override

- final field → constant

## After Learning Abstraction

✔ **Abstract**

- abstract class cannot be instantiated directly

- children must implement abstract methods
